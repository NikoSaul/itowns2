Class: setZoomLevel

setZoomLevel

new setZoomLevel(zoom, isAnimated) → {Promise}

Gets the current zoom level, which is an index in the logical scales predefined for the application. The higher the level, the closer to the ground. The level is always in the [getMinZoomLevel(), getMaxZoomLevel()] range.
Parameters:
